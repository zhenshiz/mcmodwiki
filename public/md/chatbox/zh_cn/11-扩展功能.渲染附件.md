<h2 id="a" data-toc-id="a">Attachment 渲染附件</h2>
<p>
  渲染附件是立绘的配置项之一，用于给立绘添加一些额外的渲染，比如一张图片或者一段文本。附件与立绘本身会形成一个整体，在执行动画时，附件也会跟着一起动。附件的配置方式如下：
</p><pre language="json" isclosed="false"><code class="language-json">"example": {
      "type": "texture",
      "texture": "chatbox:textures/portrait/authors.png",
      "x": 0, "y": 0, "width": 10, "height": 10, "alignX": "left", "alignY": "bottom",
      "attachment": [
        {
          "type": "text",
          "value": "Ciallo～(∠·ω&lt; )⌒★",
          "x": 3, "y": 3,
          "width": 30,
          "height": 0,
          "textAlign": "left",
          "textColor": -1,
          "lineBreak": false
        },
        {
          "只是一个提示": "可以根据需要继续添加附件"
        }
      ]
    }</code></pre>
<p>上面的<code>attachment</code> 数组就是渲染附件的配置，接下来详细介绍一下它里面每个参数的作用。</p>
<p></p>
<ul>
  <li>
    <p>
      <code>type</code>：附件的类型，目前有两种，分别是<code>texture 资源包图片</code>和<code>text 文本</code>
      。</p>
  </li>
</ul>
<div type="info" data-type="admonition" data-admo-type="info">
  <div data-type="admonition-title">INFO</div>
  <div data-type="admonition-content">
    <p>
      attachment的<code>type</code>字段与立绘的<code>type</code>字段并没有任何关联，玩家头像和物品纹理类型的立绘都是可以添加图片和文本附件的。
    </p>
    <p><span data-type="hidden-text">你问我为什么附件没有玩家头像和物品纹理的类型？因为懒得做（</span></p>
  </div>
</div>
<ul>
  <li>
    <p><code>x</code>（可选，默认0）：相对于立绘的x坐标偏移量，单位为窗口宽度的百分比。</p>
  </li>
  <li>
    <p><code>y</code>（可选，默认0）：相对于立绘的y坐标偏移量，单位为窗口高度的百分比。</p>
  </li>
</ul>
<p><strong>由于后面两种类型的字段作用区别比较大，我将分开介绍</strong></p>
<h3 id="texture-资源包图片" data-toc-id="texture-资源包图片"><code>texture 资源包图片</code>
</h3>
<p></p>
<ul>
  <li>
    <p><code>value</code>：填写图片的路径，就跟图片立绘的<code>texture</code>字段一样。</p>
  </li>
  <li>
    <p><code>width, height</code>：效果与图片立绘的同名字段相同。</p>
  </li>
  <li>
    <p>其他字段对于图片无意义。</p>
  </li>
</ul>
<h3 id="text-文本" data-toc-id="text-文本"><code>text 文本</code></h3>
<ul>
  <li>
    <p><code>value</code> : 文本内容。</p>
  </li>
  <li>
    <p><code>width</code>：文本的行宽，会影响换行和文本对齐。如果不需要换行也不考虑对齐，那这个字段以及后面两个字段都不用管。</p>
  </li>
  <li>
    <p>
      <code>textAlign</code>（可选，默认值<code>left</code>）：文本的对齐方式，可选值<code>left center right</code>。仅当<code>width</code>大于0时有效。
    </p>
  </li>
  <li>
    <p>
      <code>lineBreak</code>（可选，默认值为<code>false</code>）：文本是否换行。仅当<code>width</code>大于0时有效。
    </p>
  </li>
  <li>
    <p><code>textColor</code>（可选，默认值为<code>-1</code>，白色）：文本的颜色，采用的是ARGB格式的颜色码。
    </p>
  </li>
  <li>
    <p><s>height</s>：无意义。</p>
  </li>
</ul>
