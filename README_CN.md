# 如何给我们提供你的翻译

打开`src/languages/messages`文件夹，所有的语言文件都在这个文件夹下，比如`zh-CN.mjs`就是中文的语言文件，`en-US.mjs`就是英文的语言文件。你可以只提供一个翻译后的mjs文件，我可以帮助放到指定的位置并提供对应语言的配置。

如果你一定想要pr帮我设置好所有的配置的话，我当然也非常欢迎：

1. 在`src/assets/info/web.js`有languageList对象，你可以在这里添加你自己的语言类型，要求key是语言的Key，value则是对应语言的文字

2. 在`src/languages/messages`文件夹中添加上你对应语言的`mjs`文件

3. 在`src/languages/index.js`文件中有message属性，在这个属性中配置你的语言以及对应的语言文件

# 如何提供模组的其他语言的翻译

你可以通过`public/md/模组名/语言`的文件夹下找到对应模组的文档，你可以将文档里的内容复制到本网站的富文本编辑器中进行编辑，编辑器好后可以导出成md文件在放到对应语言的文件夹下支持网站能够展示其它语言的文档。

PS：如果你没有在`src/assets/info/web.js`中配置你的语言的话，网站也会无法切换到你的语言的文档，所以必须保证这里有配置，并希望最好也能提供对应语言的`mjs`翻译文件
